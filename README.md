# Glow Connect Stream

Application for interactive event management.

- **Check-in**: User registration and attendance.
- **Word Cloud**: Live interactive word cloud.
- **Evaluations**: Real-time feedback form.
- **Gallery**: Event photos.

## Tech Stack
- React + Vite
- TailwindCSS
- Supabase (Database & Realtime)
